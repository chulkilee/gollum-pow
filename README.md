# gollum-pow

## Installation

```bash
export repo_name="my_website"            # set host name
export repo_path="~/repos/my_website/"   # set content path (ensure '/' at end)
cd ~/.pow
mkdir "$repo_name"
cd "$repo_name"
git clone https://github.com/chulkilee/gollum-pow.git .
ln -s "$repo_path" data
```
